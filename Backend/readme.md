# Interview AI 🤖

**Interview AI** is a chatbot application designed to help users prepare for interviews.  
It analyzes resumes, generates personalized skill-based and behavioral questions, and provides feedback on answers to improve preparation.

---

## ✨ Features
- 📄 **Resume Upload**: Parse resumes to extract skills, education, and experience.
- 🎯 **Personalized Questions**: Generate technical and behavioral questions based on user profile.
- 💬 **Chatbot UI**: Interactive interface for practicing interviews.
- 🧠 **Answer Analysis**: Evaluate responses with keyword matching and semantic similarity.
- 🔄 **Adaptive Flow**: Adjusts difficulty and topics based on user performance.
